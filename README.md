<h1 align="center">Pseudowhizz</h1>
<h2>About</h2>
<p>Pseudowhizz is a free online pseudocode interpreter designed for students studying the IGCSE CIE <a href="https://www.cambridgeinternational.org/Images/697167-2026-2028-syllabus.pdf">0478</a> variant of pseudocode.
The site offers a sandbox-style environment in which pseudocode can be excecuted live, as well as an array of different tools and resources to aid students struggling with programming.</p>
